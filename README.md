# WordGen - Wordlist Generator Tool

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Flask](https://img.shields.io/badge/Flask-2.0+-green)
![Kali Linux](https://img.shields.io/badge/Kali-Linux-red)

Kali Linux üçün Python/Flask əsaslı wordlist yaratma aləti.

## Xüsusiyyətlər
- Brute Force rejimi (simvol dəsti, uzunluq, limit)
- Hədəfə özəl (leet, caps, birləşmə, suffix, tərsinə)
- Xüsusi rejim (prefix/suffix dəstəyi)
- Hydra/Medusa üçün hazır əmrlər
- Desktop-a birbaşa saxlama

## Quraşdırma
```bash
git clone https://github.com/ulvihesenov/wordgen-tool.git
cd wordgen-tool
pip install flask
python3 wordgen.py
```
